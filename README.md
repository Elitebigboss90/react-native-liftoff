# react-native-liftoff

A library to measure boot up time in the different stage for a react native app.

## Installation


```sh
npm install react-native-liftoff
```


## Usage


```js
import { multiply } from 'react-native-liftoff';

// ...

const result = multiply(3, 7);
```
